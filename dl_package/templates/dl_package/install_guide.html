インストールガイド




  1. ページ右上の「三」メニューを開き，「サインアップ」を押してアカウントを作成します．




  2. 問題がなければログインページに遷移するので，作ったアカウントでログインします．




  3. ホーム画面に遷移するので，ページ右上の「三」メニューを開き，「シリアル番号登録 & ゲームダウンロード」を押します．




  4. 購入したシリアル番号を入力して登録します．




  5. 問題がなければホーム画面に遷移するので，もう一度「シリアル番号登録 & ゲームダウンロード」を押します．




  6. お使いのプラットフォーム用のボタンを押して，ゲームをダウンロードしてください．

ご不明な点は開発チームまでお問い合わせください．

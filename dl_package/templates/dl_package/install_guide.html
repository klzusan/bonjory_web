インストールガイド




ダウンロード

  1. ページ右上の「三」メニューを開き，「サインアップ」を押してアカウントを作成します．




  2. 問題がなければログインページに遷移するので，作ったアカウントでログインします．




  3. ホーム画面に遷移するので，ページ右上の「三」メニューを開き，「シリアル番号登録 & ゲームダウンロード」を押します．




  4. 購入したシリアル番号を入力して登録します．




  5. 問題がなければホーム画面に遷移するので，もう一度「シリアル番号登録 & ゲームダウンロード」を押します．




  6. お使いのプラットフォーム用のボタンを押して，ゲームをダウンロードしてください．




インストール

  1. ダウンロードしたzip圧縮ファイルを解凍し，任意の場所に保存します．




  2. フォルダ内の「Handlime.exe」をダブルクリックして起動します．




  3. このような警告が出てきたら，「詳細情報」を押します．




  4. 「実行」を押します．




  5. このゲームはカメラを使用しますので，カメラへのアクセスを許可してください．

  2. 「Handlime.exe」をダブルクリックして起動します．

  5. 左側のリストから「プライバシーとセキュリティ」を選び，右側の設定リストを下にスクロールして「お使いのMacを保護するために...」という項目の「このまま開く」を押します．




  6. 「このまま開く」を押し，パスワードを入力して開きます．




  7. このゲームはカメラを使用しますので，カメラへのアクセスを許可してください．
